# Support

- **Bug reports:** [open an issue](https://github.com/wessamfathi/suhail/issues/new/choose) using the bug-report form — it asks for the handful of fields (Claude Code version, plugin version, OS/shell, `run_phase`) that make pipeline problems diagnosable.
- **Questions and ideas:** open a plain issue. Small project, direct channel — there is no forum or chat.
- **Security problems:** never a public issue — see [SECURITY.md](SECURITY.md).
- **Self-help:** the [README's Troubleshooting section](README.md#troubleshooting) covers the common failure modes; the artifacts under `.suhail/parts/<id>/` are the persistent record of every run step, and `.suhail/STATUS.md` shows exactly where a run stands.
